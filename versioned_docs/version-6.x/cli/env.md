id: env
title: "pnpm env <cmd>"
---

Added in: v6.12.0

Manages the Node.js environment.

## Commands

### use

Install and use the specified version of Node.js

Install the LTS version of Node.js:

```
pnpm env use --global lts
```

Install Node.js v16:

```
pnpm env use --global 16
```

## Options

### --global, -g

The changes are made systemwide.
